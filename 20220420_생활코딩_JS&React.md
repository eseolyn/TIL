36. UI vs API
UI : User Interface : 사용자가 시스템을 이용하기 위해 사용하는 것  
API : Application Programming Interface : 애플리케이션을 만들기 위해서 프로그래밍할 때 사용하는 조작장치들  
___
# React  
component : 사용자가 정의한 재사용 가능한 태그  
가독성, 재사용성이 높아지고 유지보수하기 쉬워진다.  
coding -> run -> deploy  

강사로부터 빨리 독립하는 법 중 하나 : **공식문서에 익숙해지기**  
리액트로 앱을 개발할 때 필요한 개발환경, 도구들을 모아 한번에 제공해주는 툴체인 이라는 것을 제공해준다.  
(예)Create React App, Next.js, Gatsby)  
Create React App? 그 중 single page app을 만들 수 있는 툴체인이다.  
npm : node.js를 기반으로 만들어진 여러 앱들을 명령어 환경에서 손쉽게 설치할 수 있도록 해준다. 마치 node.js계의 앱스토어 내지 구글스토어 느낌  
npx : 프로그램을 임시로 설치해서 한번만 실행시키고 지울 수 있다. 컴퓨터 공간을 낭비하지 않고 항상 최신상태로 설치한다.  

혹은 sandbox사이트를 이용해 설치없이 실행해 볼 수 있다.
